# 吼 English

Lumina is a modern UI vision for GTK 3, with GTK 4 being actively improved, created for those who seek real clarity in the Linux desktop.
Rather than reinventing everything, Lumina refines what already works in Fluent and Material You, reducing visual noise and using light, color, and motion as tools for understanding.
Lumina prioritizes visual hierarchy, readability, and everyday ergonomics. Every design decision aims to reduce cognitive friction and make the interface feel natural, coherent, and honesteven in complex applications.
This project is in active evolution. Testing Lumina, reporting issues, proposing improvements, or contributing ideas are all part of its growth. It is not a closed product, but a visual language built by the community.
Lumina is distributed under the GPL license, ensuring that the code and all improvements remain free, auditable, and shared. Anyone can study, modify, and contribute, allowing the project to grow in an open and sustainable way.
